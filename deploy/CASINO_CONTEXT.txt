 1. ¿QUÉ ES EL CASINO EXILIUM?
================================================================

El Casino Exilium es un sistema de entretenimiento interno para
los miembros de la guild Exilium en World of Warcraft. Usa una
moneda virtual llamada PandaCoins (🪙) que se puede comprar y
vender con oro del juego a razón de 1 PandaCoin = 5,000 oro.

Actualmente existe como preview visual (casino-preview.html) en
el frontend de Cloudflare Pages, completamente del lado del
cliente (HTML + CSS + JS vanilla), sin backend.

URL: https://exilium-battlepass.pages.dev/casino-preview

 3. MECÁNICA DE LA RULETA
================================================================

La ruleta tiene 10 sectores con colores alternos (dorado/verde
oscuro). El jugador selecciona una apuesta entre 4 opciones:

  • 100 🪙
  • 250 🪙
  • 500 🪙
  • 1,000 🪙

La animación dura ~5 segundos y el resultado se determina por
probabilidades predefinidas:

┌─────────────────┬──────────┬─────────────┐
│ Resultado       │ Multipl. │ Probabilidad │
├─────────────────┼──────────┼─────────────┤
│ Sin premio 😔   │   x0     │   34%       │
│ x1 🟡           │   x1     │   28%       │
│ x2 🟢           │   x2     │   20%       │
│ x3 🔵           │   x3     │   11%       │
│ x5 💎           │   x5     │    5%       │
│ JACKPOT 👑      │  x10     │    2%       │
└─────────────────┴──────────┴─────────────┘

RTP (Retorno al Jugador): ~92%


================================================================
 4. SISTEMA PANDAcoins
================================================================

  • 1 🪙 PandaCoin = 5,000 oro de WoW
  • Compra: el jugador entrega oro y recibe PandaCoins
  • Venta: el jugador vende PandaCoins y recibe oro
  • Comisión de guild en ventas: 5%
  • Bono diario gratuito: +300 PandaCoins (cada 24h)
  • Límite de giros diario: 12


================================================================
 5. LIMITACIONES ACTUALES (sólo frontend)
================================================================

  [✗] Sin persistencia — al recargar se pierde saldo, racha,
      historial y progreso
  [✗] Sin backend — compras/ventas no impactan el perfil real
  [✗] Sin autenticación — cualquiera juega sin vincular a su
      personaje de WoW
  [✗] Sin conexión al Battle Pass — el casino está aislado del
      sistema de progresión principal
  [✗] La ruleta no muestra etiquetas de premio en los sectores
  [✗] Sin medidas anti-abuso del lado servidor

 6. MEJORAS PROPUESTAS PARA INTEGRACIÓN REAL
================================================================

  ┌─────────────────────────────────────────────────────────┐
  │ 6.1 BACKEND (Worker + KV)                               │
  └─────────────────────────────────────────────────────────┘

  Endpoints necesarios en el worker (worker/casino.js):

    GET    /api/casino/balance    → saldo del jugador
    POST   /api/casino/spin       → registrar giro, actualizar
    POST   /api/casino/daily      → reclamar bono diario
    POST   /api/casino/buy        → comprar con oro (del perfil)
    POST   /api/casino/sell       → vender (sumar oro al perfil)
    GET    /api/casino/history    → historial persistente
    GET    /api/casino/jackpot    → pool del jackpot semanal

  KV Keys propuestas:

    casino:balance:{playerId}       → saldo actual
    casino:history:{playerId}       → array de movimientos
    casino:daily:{playerId}         → timestamp último bono
    casino:spins:{playerId}:{fecha} → contador de giros hoy
    casino:jackpot_pool             → pool del jackpot semanal

  ┌─────────────────────────────────────────────────────────┐
  │ 6.2 VINCULAR AL PERFIL DEL JUGADOR                      │
  └─────────────────────────────────────────────────────────┘

    • Cada miembro de la guild tiene saldo PandaCoins en KV
    • Las compras descuentan oro real del perfil
    • Las ventas añaden oro real al perfil
    • Autenticación mediante token de admin/sesión

  ┌─────────────────────────────────────────────────────────┐
  │ 6.3 INTEGRACIÓN CON BATTLE PASS                         │
  └─────────────────────────────────────────────────────────┘

    • Misiones semanales: "Gana 3 veces en la ruleta" → XP
    • Logros: "Acumula 50,000 PandaCoins" → título especial
    • Recompensas: niveles del BP otorgan PandaCoins gratis
    • Rango casino basado en acumulado histórico:

        0 - 1,000      → Novato
        1,000 - 5,000  → Apostador
        5,000 - 20,000 → Veterano
        20,000 - 50,000→ Leyenda
        50,000+        → Magnate

  ┌─────────────────────────────────────────────────────────┐
  │ 6.4 MÁS JUEGOS FUNCIONALES                               │
  └─────────────────────────────────────────────────────────┘

    🎲 Duelo de Dados
       Dos jugadores apuestan la misma cantidad, tiran 2d6,
       el mayor gana el bote menos comisión.

    📦 Cofre Misterioso
       Gastas PandaCoins para abrir un cofre con rareza
       aleatoria: Común (60%) → Raro (25%) → Épico (12%)
       → Legendario (3%). Cada rareza da premio distinto.

    🏆 Jackpot Semanal
       Pool acumulado entre toda la guild. Los jugadores
       compran tickets con PandaCoins. Sorteo automático
       cada domingo 20:00 servidor.

    🔥 Racha de Gloria
       Multiplicador progresivo por victorias consecutivas
       sin perder: x1 → x2 → x3 → x5 → x10.
